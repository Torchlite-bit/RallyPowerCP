# RallyPowerCP
RallyPowerCP keeps PallyPower's Paladin blessing/aura/seal grid exactly as it was, and adds an auto-detecting buff tracker for every other buffing class.. Log in on a Paladin and you get the full PallyPower grid. Log in on a Priest, Mage, or Druid and you instead get a matching bar that tracks *your* class's group buffs across the party/raid.
